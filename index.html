Coduri de reducere Romania

Colectie de scripturi Python open-source pentru fetch automat de coduri de reducere de la magazinele online din Romania. Datele sunt preluate live de pe Shopilo.ro, cel mai mare agregator de cupoane din Romania.

Python 3.8+ MIT License npm disponibil Date live shopilo.ro
# Instaleaza dependentele
pip install requests beautifulsoup4

# Fetch automat coduri de reducere
git clone https://github.com/shopilo-ro/cod-reducere-[magazin]
python fetch.py
